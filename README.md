# Pi Approximation Algorithm

**Overview:** Developed a C# console application that computes the value of Pi using the Gregory-Leibniz infinite series. The tool functions as a computational algorithm, allowing users to specify the number of computational iterations to demonstrate how algorithmic depth correlates with mathematical precision.

**Technical Implementations & Business Logic:**
* **Algorithmic Computation:** Implemented the Gregory-Leibniz formula utilizing a control loop to alternate between additive and subtractive fractional components to approximate the target value.
* **Dynamic Data Formatting:** Formatted the computational output to a high-precision 16-decimal floating-point structure (F16) for rigorous tabular display and data accuracy tracking.
* **Resource Management:** Handled user input parsing and execution states to dynamically scale the computation sequence based on user-defined parameters.

**Tech Stack:** C#, .NET, Computational Mathematics
